Installation
------------

You can install the package from `PyPI <https://pypi.org/project/dcpmessage/>`_ using ``pip`` or
`uv <https://docs.astral.sh/uv/concepts/projects/dependencies/>`_.

.. code-block:: shell

    # using pip
    pip install dcpmessage

OR

.. code-block:: shell

    # using uv
    uv add dcpmessage
